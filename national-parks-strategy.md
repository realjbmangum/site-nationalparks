## Part 1: Competitive Landscape Analysis

### Category 1: Official & Institutional Sources

**NPS.gov (National Park Service)**
- The source of truth for all park information
- Limited user experience, government website constraints
- No monetization (obviously)
- **Your angle:** Better UX, curated recommendations, planning tools they can't provide

**NationalParks.org (National Park Foundation)**
- Nonprofit, donation-focused
- Good content but not designed for trip planning
- **Your angle:** Trip planning focus, affiliate-friendly content

### Category 2: Directory/List Sites

**MoreThanJustParks.com** (Primary Competitor)
- Run by the Pattiz Brothers (filmmakers/photographers)
- Comprehensive content: all 63 parks ranked, state-by-state lists, printable maps
- **Monetization model:**
  - Etsy shop for maps/posters ($20-50 each)
  - Substack paid subscriptions ($7/month)
  - Affiliate links (Booking.com, Amazon guidebooks)
  - Photo/video licensing
  - Brand partnerships
- **Strengths:** Beautiful photography, comprehensive coverage, strong brand
- **Weaknesses:** Blog-style layout, no interactive map, no true directory functionality

**NationalParkBucketList.com**
- State-by-state organization
- Checklist/bucket list angle
- Good SEO for "national parks list by state"
- Less comprehensive than MTJP

**BrightStandards.com/us-national-parks-list-map**
- Downloadable checklists, printable resources
- Lead magnet focused (email capture)
- **Learning:** Printables convert well in this niche

### Category 3: Travel Aggregators

**USNews Travel Rankings**
- Authority site with "Best National Parks" ranking
- Voting/engagement features
- **Learning:** Rankings drive engagement and return visits

**Wikipedia**
- Complete list, but zero planning utility
- Your opportunity: be the "useful Wikipedia" for parks

### Category 4: Experience/Booking Sites

**shop.usparkpass.com** (Key Affiliate Partner)
- Authorized reseller of America the Beautiful Pass ($80)
- Curated park merchandise by region
- Product categories: passes, books/maps/guides, gear, apparel, stickers/pins/patches, candles, home decor
- Over $400K donated to conservation
- 25+ small business partners
- **Key insight:** They've proven the national parks merchandise market

**AllTrails**
- Trail-focused, also sells park passes
- Gear shop integration
- **Learning:** Trail data + commerce integration works

**Viator/GetYourGuide**
- Tour booking (8% commission typical)
- Park-adjacent experiences
- **Learning:** Tours convert better than gear affiliates

## Part 4: Technical Architecture

### Platform Recommendation

**Best Option: Next.js + Headless CMS**

Why:
- Static generation for SEO (fast page loads)
- Dynamic features where needed (search, filters)
- Easy to add interactive map
- Cloudflare Pages or Vercel for hosting
- Scale without server costs

**Alternative: Astro + Headless CMS**
- Even faster static sites
- Great for content-heavy directories
- React components where needed

**CMS Options:**
- Sanity.io (flexible, good free tier)
- Contentful (enterprise-ready)
- Strapi (self-hosted, free)
- Notion as CMS (simple, familiar)

### Database Schema (Simplified)

```javascript
Park {
  id: string
  name: string
  slug: string // URL-friendly: "yellowstone"
  state: string[]
  region: string
  established: date
  size_acres: number
  visitation_2024: number
  latitude: number
  longitude: number
  entrance_fee: number
  reservation_required: boolean
  best_seasons: string[]
  difficulty: enum [family, moderate, challenging, expert]
  accessibility_rating: number
  nearest_airports: string[]
  gateway_cities: string[]
  hero_image: string
  description: text
  things_to_do: string[]
  top_hikes: Hike[]
  wildlife: string[]
  weather_notes: text
  insider_tips: text[]
  affiliate_links: {
    lodging: string
    tours: string
    guidebook: string
  }
  created_at: timestamp
  updated_at: timestamp
}
```

### Interactive Map Implementation

**Recommended: Mapbox GL JS**
- Beautiful default styles
- Custom markers with park icons
- Clustering for zoomed-out view
- Popup cards on click
- Filter integration
- 50K free map loads/month

**Alternative: Leaflet + OpenStreetMap**
- Free, no API limits
- More manual styling work
- Good for budget launch

**Map Features:**
- Color-coded markers by region or difficulty
- Filter panel (state, difficulty, features)
- Click marker → popup with mini-card
- "Parks near me" geolocation
- Share filtered views via URL

### Search & Filter System

**Filters Needed:**
- State/Region dropdown
- Difficulty (checkboxes)
- Features: Camping, RV, Lodging, Wheelchair accessible, Dog-friendly trails
- Best season
- Reservation required
- Crowd level
- Distance from city

**Search:**
- Full-text search across park names and descriptions
- Algolia (premium) or local search with Fuse.js

**Mitigation:** Multiple affiliate partners, own products

**Mitigation:** Target shoulder seasons, evergreen content, email for return visits

### Risk: Copyright Issues (Photos)
**Mitigation:** Use NPS photos (public domain), Unsplash, or license properly
